- Lancez le fichier 'start.html' pour jouer
- utilisez les flèches directionnelles ou Z et S pour déplacer le vaisseau.

SI LA VIDÉO DE DÉBUT NE SE LANCE PAS :
- Allez dans le menu en haut à droite de Firefox
- Cliquez sur "Préférences"
- Dans "Vie privée et sécurité", cherchez la catégorie "Permissions"
- Cliquez sur les paramètres de la lecture automatique et sélectionnez "Autoriser l'audio et la vidéo"
- Cliquez sur "Enregistrez les modifications", rafraîchissez la page du jeu et la vidéo se lancera.
